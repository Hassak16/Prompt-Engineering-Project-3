Context-Anchored Answering (RAG Basics)

Overview:

This project demonstrates how to prevent AI hallucinations by restricting the model to a provided reference document.
The AI must answer questions using only the given context and avoid generating information that is not explicitly available in the reference text.

Features:

- Context Injection
- Closed-Book Question Answering
- Negative Constraints
- Citation-Based Responses
- Hallucination Prevention

Project Objective:

Create prompts that:
1. Use only the provided reference text.
2. Refuse to guess missing information.
3. Return "Information Not Found" when necessary.
4. Include citations for every answer.

Question:
Where does Ali live?
Answer:
Lahore
Source: Paragraph 1

Missing Information Example
Question:
What is Ali's salary?
Answer:
Information Not Found

Technologies Used in this project are:

- Prompt Engineering
- RAG Principles
- Context Anchoring
- Large Language Models (LLMs)

Learning Outcomes:

- Context control
- Factual response generation
- Hallucination reduction
- Citation enforcement
- RAG fundamentals
